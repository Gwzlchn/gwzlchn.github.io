诗集未归档……好吧就是懒了

发表于 2017-03-13 | 分类于 孙浩诗集 |

感觉对不起室友了……还是先睡觉吧~

春（其三）

雨后白云新，拂柳绕春城。
原野斑驳色，万山一律青。
和衣守煦日，闭目养安神。
尚自有闲在，清风徐叩门。

无题（其一）

旌旗日色靡，曙光暗云低。
战火燃边城，官道纷铁骑。
千里炊烟无，万户家门闭。
白骨掩荒野，征伐有功绩。

错

风声不息
叶簌簌而落
光与影的交织
绿影婆娑
我伫立
你飘然而过
或许是意外
或许早有结果

春（其一）

春回人未回，绿际掩柴扉。
借问子规鸟，远人归不归?

春（其二）

任意春风，处处点染，柳绿花红遍。
嘉音清许，人倚楼栏，风拂眼前幻。
随意雨落，池水扰乱，心中泛漪涟。
别过离分，却愿相逢，自是如何瞒。

暗夜

漆黑的夜
连星星也不剩半颗
破旧的窗
冷风摇曳着烛火
遥远的宇宙
虚无 孤寂 冷漠
粲然一笑
了结一段因果

夜色

夜色微凉
只剩下漆黑的影
月光如水
银河中沉浮的星
树的山
悄然而苍劲
是水墨的画
颜色或浅或深
让时间停滞
将失落追寻
悠悠梦醒

无题（其二）

夕阳远天平，阴云暗夜侵。
绿树泼墨色，白墙染黄晕。
垂柳拦流水，落叶逐风影。
终于全不见，璀璨万千星。

无题（其三）

悠悠潭水净，落叶舞缤纷。
亭台或楼阁，交替相掩映。
白云飞雾霭，流岚悬月明。
看惯古今事，恨昔不成行。

龙族

人世渺难尽，时事空后闻。
三峡风浪急，四海波涛平。
樱花灿若血，月色淡若星。
错乱终有定，一路负剑行。

行舟过古城

云气滃然蒸古城，烟霞浩渺裹远山。
远近诸山烟吞吐，高低众树云隐现。
山河万里江水碧，乾坤一气苍天蓝。
行舟轻快急掠过，此情此景留心间。

踏莎行

物华衰绝，风霜冷凄，茫茫晓雾白无际。雾细尘微随风去，远处朦胧仙家地。
单桨孤楫，扁舟蓑笠，醉卧山河八百里。江水浩荡湍流急，限此狭道奈何意。

莽林

苍茫竹林独远行，常涌绿浪叶飘零。
偶有鸟声无处觅，天涯何处是归程。

如梦令

风细杨柳堆烟，
坐卧香草河畔。
清流蕴碧天，
丽日拂暖慵懒。
人影散漫，
远外数峰青山。

# 孙浩 # 诗集
