配置Atom

  1. 安装插件

    Ctrl+shift+P输入settings调出设置页，在左侧选项卡有Install选项，搜索以下项目并安装

    • latex
    • language-latex
    • pdf-view
    • activate-power-mode(可选，用于美化)

  2. 配置Atom

    在Packages选项卡中找到Latex 插件进行设置

    1. Tex Path 中找到你安装Texlive的目录下bin文件夹

      D:\texlive\2016\bin(这是我的目录)

    2. SumatraPDF Path 中找到你安装SumatraPDF 的目录

      通常点击一下输入框就会自动搜索到

  地处盆地，势起东南。东西两向，环围青山。川流时继，人事安闲。四季风情，朗月清天。浮云换日，物华不迁。  

  春来物阜，芳馨遍田。荠麦黄花，春意盎然。雀鸟呼应，燕子飞还。碧苔次第，桃花点染。万物复苏，芳草沿岸。清流荡漾，鱼游戏潜。丽日初升，朝霞始现。薄雾轻飞，气润微甘。野披轻纱，山绕白练。雾凝露起，珍珠四散。翠叶凝珠，丹蕊微汗。

  夏及丰茂，白日气炎。杂草群丛，道路皆掩。高树叶盛，热风鸣蝉。叶繁影碎，光点星斑。绿野盈目，心旷神安。逢有闲暇，出行游玩。青葱郁盛，垂柳如烟。折叶飞旋，截卢射远。大雨有几，阶院皆漫。湿花落圃，垂叶半偏。流云滞雨，雨过晴天。斜荷半倾，菡萏竞艳。水清潺潺，细流涓涓。戏虾游鱼，小蟹河蚌。竹篓提携，天色将晚。鸡鸣犬吠，白鹤飞散。

  秋临衰颓，满目阑珊。千山旷远，空林叶遍。虫声起伏，雁鸣长传。独钓秋水，影映万千。竿随波动，景使人幻。四下无人，秸秆堆山。萧风飒起，江天相连。独立小院，林梢月弦。黑影重重，清辉更惨。松柏益苍，草木凋残。负道径上，青石巉岩。山道积叶，谁人可怜。

  冬至寂寂，长夜漫漫。霜重雾浓，物仅点线。日出东方，茫茫一片。路尽枯草，人行车碾。凹凸不平，时运多舛。纵此何妨，终上云端。难得大雪，洋洒八面。天地无垠，江山色变。银装素裹，妖娆画卷。宣纸铺好，墨笔谁添。青松翡翠，冰棱晶钻。岭戴絮帽，简洁寥宽。

  今日相思，续续不断。夭下之事，难得两全。当下明见，免后遗叹。
